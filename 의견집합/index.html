웹 페이지의 목적

부산에 거주하면서 버스를 타야 하는데 버스의 운행시간, 버스가 지나가는 정류장의 위치를 찾는데 도움을 주기위해서
이 페이지는 다양한 기능을 제공해드립니다.


웹 페이지에서 제공하는 기능

메인 화면화면의 상단에는 '다양한 버스 정보', '시외버스사이트', '도움말', '평가' 버튼이 존재합니다.
아래의 설명을 읽고 필요한 기능의 버튼을 눌러 해당 페이지로 이동하세요.

다양한 버스 정보
해당 버튼을 클릭시 다양한 버스 정보 페이지로 이동하게 됩니다.
시내버스노선 페이지에서는 api와 javascript를 이용해서 부산에 존재하는 정류장 정보를 입력하면
관련된 정류장의 정보들을 표시합니다.
또한 부산의 모든 시내버스의 운행시간이 담긴 엑셀 파일을 다운로드 할 수 있으며
해당 엑셀 파일의 매크로 기능을 이용하면 원하는 버스 번호의 행 위치로 이동 할 수 있습니다.
(자세한 사용법은 '도움말'을 이용해주십시오)

시외버스예매사이트
시외버스를 모바일로 예매해주는 사이트로 이동합니다.
링크 주소는 'https://www.bustago.or.kr/'로 신뢰 가능한 사이트이니 걱정 할 필요는 없습니다.
추가적으로 현 메인 페이지의 하단에 시외버스와 관련된 웹 페이지로 이동할 수 있는 로고가 존재하며
부산시외버스노선, 운행정보에 대한 정보가 필요하시다면 '부산종합버스터미널주식회사'라고 적힌 로고를 클릭하십시오.
(자세한 내용은 '도움말'을 이용해주십시오)

도움말
웹 페이지의 기능중 사용법이 어려운 기능에 대해 상세한 설명을 첨부하여 도움을 드리는 기능입니다.
정류장 조회 방법, 시외버스예매방법, 부산시외버스노선에 대한 정보를 찾는 방법, 엑셀 파일에서 매크로를 사용하는 방법 등을 알려드립니다.
만약 도움말에서 부족한 부분이 있다고 느끼신 다면 '평가'페이지에서 의견을 남겨주십시오.

평가
웹 페이지의 전체적인 만족도를 조사하기 위한 질문들을 첨부하였습니다.
개선점을 생각하기 위한 정확한 평가를 부탁합니다.
추가하면 좋을 것 같은 기능, 도움말에서 부족하다고 느꼈던 점 등 다양한 의견을 남겨주시면 반영하려고 노력하겠습니다.
